About Me

Hello! I'm Pawan Bhati, a dedicated student currently pursuing a Bachelor of Computer Applications (BCA). My passion for technology and programming has driven me to explore various areas within the field.

One of my recent projects is a Book Recommender System, which combines machine learning and Flask to provide personalized book recommendations. This project leverages machine learning algorithms to analyze user preferences and suggest books accordingly, while Flask handles the web application's backend, ensuring a smooth and efficient user experience.

The dataset used for this project was sourced from Kaggle, featuring a comprehensive collection of over 1,000,000 books. From this vast dataset, we focused on selecting 50 books based on user reviews and popularity. This careful curation ensures that the recommendations are not only relevant but also based on real user feedback and book popularity.

Anyone interested can explore this dataset and see how our recommendations are derived. Through this project, I've honed my skills in both machine learning and web development, learning how to integrate complex algorithms with user-friendly web interfaces. My goal is to continue building innovative solutions and expanding my knowledge in these exciting fields.
